Политика конфиденциальности

Дата вступления в силу: 26 октября 2024 года

1. Сбор информации

YouTube Analytics Dashboard собирает следующую информацию через YouTube API:

  • Статистику подписчиков (прирост и отток)
  • Базовую информацию о канале (название, количество видео)
  • Аналитику просмотров и вовлеченности

2. Использование информации

Собранная информация используется исключительно для:

  • Отображения статистики в личном кабинете пользователя
  • Формирования отчетов и аналитики
  • Предоставления функциональности приложения

3. Хранение данных

Мы не храним ваши данные на наших серверах. Вся обработка происходит:

  • Локально на устройстве пользователя
  • В режиме реального времени через YouTube API
  • Без постоянного сохранения личной информации

4. Передача данных

Мы не передаем ваши данные третьим лицам. Вся информация:

  • Остается конфиденциальной
  • Используется только для работы приложения
  • Не продается и не сдается в аренду

5. Безопасность

Мы принимаем разумные меры для защиты вашей информации:

  • Используем OAuth 2.0 для безопасной аутентификации
  • Работаем только с официальными YouTube API
  • Не запрашиваем избыточные разрешения

6. Ваши права

Вы имеете право:

  • Отозвать доступ к вашим данным в любой момент
  • Запросить удаление любой сохраненной информации
  • Получить информацию о том, какие данные обрабатываются
